```
Usage: index [options] <url>

Options:
  -V, --version                      output the version number
  -d, --domains <list>               comma-separated list of domains
  -e, --extensions <list>            comma-separated list of extensions (default: "css,html,js")
  -o, --output <dir>                 path to output directory (default: ".")
  -s, --sort-every <int>             sort output files every x seconds, removing duplicate values (default: 60)
  -x, --proxy <[proto://]host:port>  use proxy for chromium
  -h, --help                         display help for command
```
